A beta build of Floe v2 - for testers, early adopters, and anyone wanting to try in-development features ahead of the full release.

{/* truncate */}

![Floe v2.0.0-beta.2](/images/screenshots/overview.png)

:::warning Beta release
Expect rough edges and bugs. If you'd rather play it safe, stick with the latest stable release - all of this will land there once v2 is finished. More info on [beta testing here](/docs/about-the-project/beta-testing).
:::

What's new in v2 so far:

- New 2-page UI layout, plus a distraction-free PERFORM page
- Creative randomisation strip for exploring preset variations
- Granular synthesis (Playback and Fixed modes)
- Per-layer arpeggiator with sliced-loop playback
- Visual filters, EQ and LFO with draggable nodes
- More natural knob/slider response on frequency, ratio, threshold and gain params
- Per-effect bypass buttons and unified Mix knobs on the FX rack
- New Stereo width controls per-layer, plus Balanced and Bass Mono widen modes
- New LFO shapes: Random Steps, Random Glide, Pluck, Pluck Sharp, Pulse Narrow, Pulse Wide, Trapezoid
- 3-band EQ with new LP/HP types, and EQ as an FX rack effect
- Instance Config panel for reproducible performances (transport reset, keyswitch, seed)
- MIDI CC assignments display panel
- Improved legacy-parameters system with visual indicators and a modernise flow
- New `.floe-pkg` package format, plus encrypted `.floe-pkg-enc` for developers
- Undo/redo for all audio-state changes
- Right-click copy/paste/reset menus throughout the UI
- Streamlined browsers with collapsible cards
- New Modern compressor algorithm alongside the Vintage one
- GUI and audio-engine performance improvements
- Crisper-looking UI throughout
