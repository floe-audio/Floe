Floe 1.1.2 is out now with an important fix for Windows users and various stability improvements across all platforms.

<!-- truncate -->

The big news here is that text input on Windows actually works properly now. Previously, typing into Floe would often trigger your DAW's transport controls instead of entering text - so you'd be trying to search for a preset and accidentally start playback. Very annoying! This has been one of the most requested fixes, and it's now sorted.

Besides that, this update cleans up quite a few edge-case issues across all systems, making everything more stable and polished.

Download Floe 1.1.2 from the [download page](/download). See the full [changelog](/docs/changelog) for all changes.
