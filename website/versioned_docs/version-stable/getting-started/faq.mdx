# Frequently asked questions

### Can I add my own samples into Floe?
Floe does not support ad-hoc sampling at the moment. We describe Floe as a _sample library platform_ rather than a sampler (alternatively you could describe it as an open ROMpler). It is a platform for comprehensive, polished products. If you are a developer, you can use our Lua tools to make your own sample libraries for Floe.

### Does Floe load SFZ/Soundfont/other formats?
No, Floe currently only loads the Floe sample library format, an open format with similar ideas to SFZ but using the Lua programming language. Floe also supports the legacy Mirage sample library format used by some FrozenPlain libraries.
