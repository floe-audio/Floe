# Randomisation

> How to use Floe's randomisation features to explore new sounds

Floe has various randomisation features to help you explore new sounds quickly.

![Floe's main user interface with numbered overlays indicating the randomisation features.](/images/whole-gui.png)234

1

-   **1**: Browser controls. Each [browser](/docs/key-concepts/browsers) has a set of controls to navigate through the current item list or load a random one. These buttons are also available on the browser itself. In this case, these controls are for the Presets browser. The left and right arrows navigate to the previous/next item in the results list. The random icon loads a random item from the results list. Open the browser to see the current results list and choose different filters to change or narrow down the results. For example, you can open up the browser and select the filter for the library 'Music Box Suite'. The random button will now only choose presets relating to that library.
-   **2**, : Menu for _Randomise All Parameters_ function. This completely randomises all the parameters in Floe and loads random instruments and impulse response; these both honour the currently selected filters in the browsers.
-   **3**: Layer instruments randomisation. Randomise all 3 layers' instruments at once. It's as if you clicked the random button in each layer's instrument browser one after the other. It honours the currently selected filters in each layer's instrument browser.
-   **4**: Effects randomisation. Entirely randomise all the effects: change all parameters, reorder and enable/disable effects. This is a quick way to explore new effect combinations and settings.
